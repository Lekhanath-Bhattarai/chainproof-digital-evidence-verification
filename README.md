# ChainProof: Secure Verification of Digital Forensic Evidence

ChainProof is a Practical Cryptography coursework project focused on protecting the integrity and authenticity of digital forensic evidence using cryptographic techniques such as hashing, digital signatures, certificates, and public key infrastructure (PKI).

## Features

- RSA key generation
- Certificate-based authentication
- SHA-256 evidence hashing
- Digital evidence signing
- Signature verification
- Tamper detection
- Evidence integrity validation
- Certificate validation
- Attack simulation and mitigation

## Technologies

- Python
- Flask
- Cryptography Library
- RSA
- SHA-256
- X.509 Certificates
